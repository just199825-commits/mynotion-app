# MyNotion v2 ⚡

Je persoonlijke productivity workspace met 9 modules.

## Modules
- 🏠 **Dashboard** — Alles in één oogopslag
- ✅ **Habit Tracker** — Grid + heatmap, streaks
- 📋 **Taken** — Kanban, subtaken, prioriteiten, gebieden
- 📁 **Projecten** — Eigen pagina's, links, braindump, gekoppelde taken
- 📝 **Notities** — Markdown, tags, zoeken, pinnen
- 🎯 **Goals** — Korte + lange termijn, categorieën
- ⏱️ **Pomodoro** — Custom timer, stats, koppel aan taken
- 💰 **Budget** — Transacties, categorieën, spaardoelen
- 📓 **Journal** — Mood tracker, prompts, dagboek

## Features
- 🌙 Dark/Light mode toggle
- 💾 Data opslag via localStorage
- 📱 Sidebar navigatie (inklapbaar)

## Deploy op Vercel (gratis)

1. Upload alle bestanden naar een GitHub repository
2. Ga naar [vercel.com](https://vercel.com) → importeer het project
3. Klik Deploy → klaar!

## Lokaal draaien

```bash
npm install
npm run dev
```
